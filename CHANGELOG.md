# Changelog

All notable public changes to QuantaDesk will be documented in this file.

## [0.1.0] - 2026-06-01

### Added

- Initial clean-history public repository for QuantaDesk.
- Static project page for GitHub Pages.
- Public roadmap, usage guide, publishing guide, and security-boundary notes.
- Publication checklist for keeping private strategy material out of the public
  repository.
- Bug report and feature request templates.
- Public-safety GitHub Actions scan for sensitive keywords and private-path
  patterns.
- Apache-2.0 license and public NOTICE file.

### Security

- Private strategy implementations, parameter sets, proprietary research,
  market lists, operational candidates, credentials, journals, and operator
  state remain outside the public release.

### Next

- Local operator console.
- Backtest and report kit with sample data and educational toy strategies.
- Private strategy plug-in interface.
